Getting Started

All source code can be found at:
https://github.com/scala-vienna/DevFest2013-Workshop/.

Just clone it via:
> git clone https://github.com/scala-vienna/devfest2013
Or just download ZIP file from here if you don't have git

Project uses Typesafe Activator template, thus you can develop in the web browser or any other IDE which Activator supports. We will be developing in browser during this workshop, but if you are interested to code in your favourite IDE, please refer to Activator documentation about the topic.

Develop in Web Browser

To develop in web browser just start following script from your cloned project directory:
> ./activator ui

On Windows please start:
> ./activator.bat ui

This may take a little while Activator to start up and dovnload all the dependencies, so please be patient. After a while Actiator will open your favourite browser automaticaly.

If the Activator doesn't open anything please refer to:
http://localhost:8888/app/DevFest2013-Workshop

We wish you a happy coding...

Project Structure and Code Organization

All source code needed for our workshop can be found in: /src

Production code can be found in: /src/main/scala

Test code can be found in: /src/test/scala

All the other directories within the project are irrelevant for the purpose of the workshop. They serve to build system and testing framework tasks.
